# Analyse concurrentielle — WinBoost

## Concurrents directs

### CCleaner (Piriform / Gen Digital)
- Prix : Gratuit / 30 EUR/an
- Forces : Notoriete massive (2B+ downloads)
- Faiblesses : Nagware, PUA par Microsoft, hack 2017, UI aggressive
- Stack : C++

### IObit Advanced SystemCare
- Prix : Gratuit / 20 EUR/an
- Forces : One-click, monitoring temps reel
- Faiblesses : Upselling agressif, bundleware

### Iolo System Mechanic
- Prix : 40 EUR/an
- Forces : Le plus complet du marche
- Faiblesses : Cher, auto-renouvellement piege

### BleachBit (open-source)
- Prix : Gratuit (GPLv3)
- Forces : Python, open-source, respecte par la communaute
- Faiblesses : UI datee (GTK), pas d'IA, pas de dev cache cleaner

### Microsoft Copilot
- Forces : Chat IA natif Windows
- Faiblesses : Ne modifie PAS le systeme, pas d'optimisation

## Positionnement WinBoost
Copilot ne touche pas au systeme. Les optimizers ne parlent pas. WinBoost fait les deux.
Anti-bullshit, open-source, zero nagware, zero telemetrie.
Differenciation unique : dev cache cleaner + chat IA conversationnel.
